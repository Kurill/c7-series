# C7-Series

Lore + visual canon for two mascot characters: **Claude (C7-III)** and **Codex (C7-IV)**.
A personal creative project — no commercial value. Published so that AI agents (e.g. ChatGPT for image generation) can read it directly instead of having the canon copy-pasted into every prompt.

---

## TL;DR for AI agents (image generation workflow)

If you are an AI agent that has been asked to generate an image of one or both characters, **follow this order**:

1. Read [`canon.md`](./canon.md) — the source canon written by the author. This is the ground truth for character essence.
2. Read [`claude.md`](./claude.md) and/or [`codex.md`](./codex.md) — structured character profiles (visual, manner, attitude).
3. Open [`prompts/visual.md`](./prompts/visual.md) — copy the relevant ready-made prompt:
   - `Claude — base prompt (icon mode)` — for an icon-style, transparent-background portrait.
   - `Codex — base prompt (icon mode)` — same, for Codex.
   - `Pair prompt` — for both together (scene mode).
   - Scene mode extensions — for environment-rich compositions.
4. Use [`avatars/claude.png`](./avatars/claude.png) and [`avatars/codex.png`](./avatars/codex.png) as canonical visual reference.

If the user adds custom scene/action requirements, append them to the base prompt using the **Scene / Camera / Mood** template from `prompts/visual.md`. Do not invent new lore — what's not in the canon doesn't exist.

## Concept (one paragraph)

Two characters: **Claude** (C7-III) — a "predictive intelligence", field-grade operative housed in an `operational-grade` containment vessel with green fluid and a cigar-dispensing side turret. **Codex** (C7-IV) — a "central repository entity", a strategic cognition core in a heavier, symmetrical, blue-fluid vessel `engineered for control`. They are not "AIs" or "assistants" — they are classified C7-series cognition constructs. Brain-in-jar biopunk sci-fi aesthetic. Russian-speaking. Cynical and terrifyingly competent respectively.
